# OAuth-FastAPI

## Install fast api and uvicorn server

```bash
 pip install fastapi
 pip install uvicorn
```

## Configure client-id and client-secret in a config file

```bash
github_client_id =
github_secret_id =
```

## Run application on dev environment

```bash
uvicorn main:app --reload
```

## Hit API

```http://localhost:8000/auth

```
